utils.save_confusion_matrix_plot(y_true, y_pred, labels, save_path, metrics=None, title='Confusion Matrix')[source]

Save a confusion matrix plot to a file using sklearn’s ConfusionMatrixDisplay.

Parameters:

  • y_true (ndarray`) – Ground truth labels.

  • y_pred (ndarray`) – Predicted labels.

  • labels (list) – List of class labels.

  • save_path (Path) – path to save the confusion matrix plot.
